# 水野班環境
水野班の環境周りの情報共有用リポジトリです.  

## Content
- SetUp  
    - 各種環境の立ち上げ  
    - 一つで完結する内容が主  
- HowTo  
    - 各種環境の利用方法  
    - SetUpの内容をまとめたフローなどもこちら
